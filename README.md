## Projects

**[NursingSimulation](https://github.com/krveg55/NursingSimulation)**
Real-time simulation platform for nursing education. Multi-project .NET solution: ASP.NET Core REST API, SignalR hub for live session updates, EF Core with SQL Server, Blazor Server frontend, and a WPF desktop client. Teachers can add observations and vitals during a running simulation and all connected clients update instantly.

**[Solarsystem](https://github.com/krveg55/Solsystem)**
Solar system simulation with real orbital mechanics. Built in C# with WinForms. Planets and moons subscribe to a simulation clock and update their positions each tick using trigonometry. Supports adjustable speed, zoom views, and retrograde orbits. The physics model lives in a separate class library from the GUI.

**[Event Registration System](https://github.com/krveg55/oblig5)**
Full-stack registration and login system in Java/Spring Boot with PostgreSQL. Passwords are secured with PBKDF2WithHmacSHA1 + random salt, implemented from scratch rather than delegated to a library. Includes a JUnit 5 test suite covering input validation.

**[Traffic Light Simulation](https://github.com/krveg55/Lyskryss)**
Traffic intersection simulator in C++/Win32. Cars spawn probabilistically, obey a four-state light cycle, check for collisions with each other, and commit to crossing once they pass the stop line. Spawn probabilities are tunable at runtime via keyboard.
